Bem-Vindo ao CertAuto

Para criar um certificado para um unico aluno use o comando

python certAuto.py "Nome Aluno" "Brasília, 04 de maio de 2017"

Para criar certificados baseando em uma lista, de nome use o comando
anterio no local do nome informa o arquivo txt da lista

python certAuto.py "participantes.txt" "Brasília, 04 de maio de 2017"
